Welcome to DER Analysis

This is an application for DER visualization and analysis using data from BDGD's electricity distribution networks. Use the available filters to personalize your map experience.

You can explore the map with geospatial data and apply filters like Utility, Substation and Circuit to refine the results.

Go to Mapa

Application Features

  • Visualization of energy distribution networks on the map.
  • Dynamic filters for Distributors, Substations and Circuits.
  • Visualization of the geographic location of network elements.

How to use

  • Go to the map section.
  • Select the desired filters from the side panel.
  • Select the scenarios.
  • View the power distribution networks on the map and the voltages and currents on busses.

About

The platform allows the user to view a graphical representation of the electric network, its geographic distribution, as well as electrical quantities such as busses voltages and currents and powers transformers This application was developed using OpenLayers, Flask and dynamic SQL Server database queries.
