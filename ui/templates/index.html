Bem-vindo ao App de Mapas

Esta é a sua aplicação para visualização e análise de segmentos de retas baseadas em coordenadas geoespaciais. Utilize os filtros disponíveis para personalizar sua experiência no mapa.

Você pode explorar o mapa com dados geoespaciais e aplicar filtros como Distribuidora, Subestação e Circuito para refinar os resultados.

Ir para o Mapa

Recursos da Aplicação

  • Visualização de segmentos de reta no mapa
  • Filtros dinâmicos para Distribuidora, Subestação e Circuito
  • Estilização personalizável das camadas do mapa

Como usar

  • Navegue até a seção de mapa
  • Selecione os filtros desejados no painel lateral
  • Visualize os segmentos de reta no mapa

Sobre

Esta aplicação foi desenvolvida para facilitar a visualização e interação com dados geoespaciais usando OpenLayers, Flask e consultas dinâmicas a banco de dados SQL Server.
